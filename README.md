# Netflix Clone

A simple Netflix Landing Page Clone built using HTML and CSS.
This project recreates the basic layout of the Netflix homepage including the hero section, feature sections, FAQ area, and footer.

The goal of this project was to practice frontend layout design, styling, and responsive page structuring similar to a real-world streaming platform interface.

# Features

🎥 Netflix-style Hero Section

🔐 Sign In button navigation bar

📺 Feature sections such as:

Enjoy on your TV

Download shows to watch offline

Watch everywhere

Kids profile section

❓ Frequently Asked Questions (FAQ) section

📧 Email input for membership start

🖤 Netflix themed dark UI design

📱 Basic responsive layout structure

# Built With

HTML5

CSS3

This project focuses on building a visually structured UI using only basic frontend technologies without any frameworks.

📚# Learning Purpose

This project helped in practicing:

Page layout structuring

CSS Flexbox

UI cloning of real websites

Styling and component arrangement

Building a full landing page from scratch
